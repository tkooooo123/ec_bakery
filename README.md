## 安裝流程
1. 開啟終端機，執行以下指令：
 ```
 $ git clone https://github.com/tkooooo123/ec-lihouse.git
 ```
2. 進入專案資料夾:
```
 $ cd ec_bakery
```
3. 使用npm安裝套件
```
$ npm install
```
4. 新增.env檔案，設定環境變數
```
VITE_BASE_URL=https://lit-lowlands-54861.herokuapp.com/api/

```
5. 執行專案
```
$ npm run dev
```
